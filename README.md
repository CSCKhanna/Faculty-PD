# URI Faculty PD Training Calendar

This repository is a GitHub Pages test wrapper for the URI Faculty Development Training Calendar.

The current `index.html` embeds the hosted demo page at:

https://mghp.osn.xsede.org/emri/faculty-training-webpage/index.html

Notes:

- The embedded site is static and can be viewed through GitHub Pages once Pages is enabled.
- The `Update now` button inside the embedded site still requires the Node server/API version of the app. Static hosts such as GitHub Pages cannot run that server endpoint by themselves.
- A full static package is available locally in the Codex workspace at `outputs/faculty-training-webpage/`.
